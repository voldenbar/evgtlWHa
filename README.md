# 项目简介

## 核心功能
- 基于 YOLOv8 的安全防护佩戴检测系统
- 支持 12 类防护装备检测：手套、护目镜、头盔、口罩、防护服、鞋子等
- 提供完整源码、训练好的模型、30,000 张标注数据集
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示检测结果：目标位置、目标总数、置信度等信息
- 支持批量检测功能，可查看每一张检测结果
- 提供自定义配置选项：置信度阈值、IOU阈值、显示标签和原图选择
- 包含完整的 UI 界面，支持模型选择功能

## 适用场景
- 工业安全监控：实时检测工人是否按规定佩戴安全防护装备
- 建筑施工现场：自动检查头盔、安全服等佩戴情况
- 医疗机构：监控口罩等防护用品的佩戴情况
- 实验室环境：检测护目镜、手套等防护装备的使用
- 安防监控系统：自动识别未按规定佩戴防护装备的人员

## 技术特点
- 采用 YOLOv8 深度学习框架进行目标检测
- 环境配置：Python 3.9 + OpenCV + PyQt5 + Ultralytics + PyTorch 1.9
- 提供完整的项目文件结构，包括程序源码、UI界面文件、数据集和教程
- 训练数据集包含 30,000 张标注图片，覆盖 12 类安全防护装备
